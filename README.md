# cateye
Download from Kamu Dev and Install tool.

## Install Cateye

```zsh
brew tap DiamondGotCat/cateye
brew install cateye
```

## Install or Update Kamu Package With Cateye

```zsh
sudo cateye install [package name]
```

## Update Cateye

```zsh
sudo cateye update
```

## Catest (Cateye Testing File)

```zsh
sudo cateye install catest
```
